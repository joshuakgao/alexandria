# 3DGS-CD: 3D Gaussian Splatting-based Change Detection for Physical Object Rearrangement

![[2025-3dgs-cd-thumbnail.png]]

## Research gap

3DGS-CD is the first 3D Gaussian Splatting-based method for detecting physical object rearrangements in 3D scenes. Given pre-change images (used to train a 3DGS model) and potentially sparse post-change images (as few as one), it detects object-level 3D changes — removals, insertions, and movements — including 3D segmentation masks and 6D pose change estimation for moved objects.

## Contributions

- First 3DGS-based 3D change detection method, exploiting 3DGS's real-time rendering for efficient pre/post-change comparison
- Works with sparse post-change inputs (as few as one image), no depth sensors, no pre-defined object classes, and no user prompts
- Object-level change classification (moved/removed/inserted) via 2D-3D feature correspondence matching
- 6D pose change estimation for moved objects with render-and-compare refinement
- Three orders of magnitude faster than NeRF-based alternatives with higher accuracy
- Downstream applications: object reconstruction from change masks, robot workspace reset, incremental 3DGS model update

## Method

**Pre-change 3DGS**: Standard 3DGS training on pre-change images via SfM + Gaussian optimization.

**Camera Localization**: Post-change cameras localized against pre-change SfM point cloud via standard visual localization.

**2D Change Detection**: Render pre-change RGB at post-change viewpoints. Compare EfficientSAM embeddings (cosine similarity) between rendered and captured images. Otsu thresholding → change masks. Query EfficientSAM with bounding boxes on both images → move-out masks (on rendered) and move-in masks (on captured), filtered by confidence > 0.95.

**Object Association**: Back-project move-out masks using 3DGS-rendered depth → per-view partial point clouds. Associate across views via Hungarian matching on Hausdorff distance, verified by EfficientSAM embedding cosine similarity (>40%). Match object templates to move-in masks via SuperPoint 2D-3D correspondences → classify as moved (sufficient inliers), removed (no inliers), or inserted (unmatched post-change segments).

**3D Segmentation**: Project object point clouds onto pre-change images → query EfficientSAM for additional masks → multi-view voxel voting (majority >95% threshold).

**Pose Change**: RANSAC-PnP on 2D-3D correspondences for initial estimate → render-and-compare refinement minimizing photometric error between transformed 3DGS renders and post-change captures.

## Datasets & evaluation

Up to 14% higher F1 accuracy than NeRF-Update (SOTA NeRF-based method). Three orders of magnitude faster runtime (excluding 3DGS training). Works reliably with as few as 1-3 post-change images. Validated on public datasets and self-collected real-world scenes with cluttered tabletop environments. Downstream applications demonstrated: object reconstruction via change-mask-guided 3DGS extraction, robot workspace reset by detecting displaced objects, and 3DGS model update by transforming Gaussians within detected change masks.

## Limitations

- Requires pre-change 3DGS training (minutes to hours); not suitable for truly online operation unlike Online SCD (2026)
- Assumes most of the scene is unchanged for post-change camera localization; large-scale changes may break localization
- No semantic understanding of changes — objects are detected purely by geometric/appearance differences, not labeled
- Object association via Hausdorff distance and embedding similarity may fail with very similar objects or heavy occlusion
- 6D pose estimation limited to rigid body transformations; non-rigid deformations not handled
- EfficientSAM segmentation quality limits change detection accuracy, especially for small or highly occluded objects

## Key takeaways

The pipeline leverages 3DGS for novel view synthesis and EfficientSAM for zero-shot segmentation: (1) Train a 3DGS on pre-change images; (2) Localize post-change cameras against the pre-change SfM point cloud; (3) Render pre-change RGB-D at post-change viewpoints and compare EfficientSAM embeddings for 2D change detection; (4) Associate detected 2D object segments across views using point cloud proximity and embedding similarity (Hungarian matching); (5) Fuse multi-view masks into 3D object segmentation via voxel voting; (6) Estimate 6D pose changes for moved objects using RANSAC-PnP on 2D-3D feature correspondences; (7) Refine poses via render-and-compare optimization.

The method achieves up to 14% higher accuracy and three orders of magnitude faster performance than the SOTA NeRF-based change detection method, enabling practical downstream applications: object reconstruction, robot workspace reset, and 3DGS model update.

3DGS-CD occupies a distinct niche in the change detection topic — it focuses on **object-level physical rearrangement** (moved/removed/inserted objects) rather than the scene-level semantic change detection of other papers. Where GeSCF (2024) and Online SCD (2026) detect pixel/region-level changes for place recognition or surveillance, 3DGS-CD produces object 3D segmentation masks and 6D pose changes — outputs directly useful for robotic manipulation.

Compared to SceneDiff (2026), which also detects object-level changes in multi-view scenes, 3DGS-CD uses 3DGS as the scene representation (enabling efficient rendering at arbitrary viewpoints) rather than computing changes from raw image pairs. SceneDiff is training-free and uses π³ + SAM + DINO, while 3DGS-CD requires pre-change 3DGS training but works with sparser post-change input.

The EfficientSAM embedding comparison for change detection connects to the foundation-model-based approach used across this topic (SAM in GeSCF, SAM+CLIP in DynamicEarth, DINO in SceneDiff).
