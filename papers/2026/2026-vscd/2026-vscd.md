# VSCD: Video-based Scene Change Detection in Unaligned Scenes

![[2026-vscd-thumbnail.png]]

## Research gap

VSCD introduces a new task formulation for change detection: given two unconstrained RGB videos of the same indoor environment recorded at different times, predict pixel-wise change masks for each query frame. Unlike prior image-based SCD (which assumes mild viewpoint variation) or video-based abandoned object detection (which assumes constrained camera trajectories with few changed objects), VSCD handles unconstrained camera motion, severe cross-view misalignment, no temporal synchronization, and many simultaneous object-level changes.

## Contributions

- New task formulation (VSCD) capturing realistic unconstrained video-based change detection with many simultaneous object changes — filling the gap between image-pair SCD and constrained video AOD
- Large-scale benchmark: 1.1M+ synthetic frames across 218 environments with pixel-accurate change masks, plus real-world test set for sim-to-real evaluation
- VSCDNet architecture with frame-level alignment, patch-level correspondence, and dual confidence fusion (frame + patch level) for robust multi-reference change detection
- Real-world robotic deployment on Stretch 3 for visual surveillance and object incremental learning

## Method

**Stage 1 — Frame-level alignment:** Uniformly sample keyframes from both videos, encode with frozen SAM-ViT, compute cosine similarity grid refined by a shallow convolutional head. Soft matching distribution selects reference candidates per query frame. Matched segment proposals group temporally consistent matches.

**Stage 2 — Patch-level correspondence:** For each query-reference pair, compute local correlation volumes in a k×k window, apply softmax to get patch matching distribution, compute expected displacement field, and warp reference features via bilinear sampling. Extract compact change features from query and warped reference.

**Stage 3 — Confidence-weighted fusion:** Combine frame-level confidence (from soft matching) and patch-level confidence (from matching distribution peakiness and entropy) to weight per-candidate change features. Fuse into a single representation per query frame. Decode to full-resolution change mask with query RGB injection for fine boundary recovery. Trained with BCE + soft Dice loss.

## Datasets & evaluation

Overall synthetic F1: 36.6% vs. next best GeSCF 29.5%. Overall real-world F1: 25.4% vs. GeSCF 17.3%. Performance improves with more changed objects (32.1→39.0 across low-to-high), demonstrating robustness to complex scenes. Ablation shows frame-level confidence (C_f) is most critical; patch-level confidence (C_sp) provides complementary spatial reliability. Alignment token adds +1.1 F1. All confidence components together contribute +1.8 F1 over no-confidence baseline.

## Limitations

- Real-world F1 (25.4%) significantly lags synthetic (36.6%), indicating a substantial sim-to-real gap remains
- Training only on synthetic data; real-world fine-tuning or domain adaptation could improve transfer
- Currently limited to indoor environments; outdoor VSCD with larger scale variation and weather changes is unaddressed
- No explicit 3D reasoning — could combining VSCD's temporal video matching with 3D reconstruction (VGGT-CD, O-SCD) yield better results?
- Robotic deployment results are preliminary (62.1% hit rate for surveillance, 2/4 objects for incremental learning)
- Object relocation treated as two independent changes (disappearance + appearance) rather than tracked as a single event

## Key takeaways

To study this setting, the authors build a large-scale benchmark with over 1.1 million synthetic frames (from AI2-THOR and Unreal Engine) across 218 environments, plus 8 real-world video pairs for sim-to-real evaluation. They propose VSCDNet, a query-centric multi-reference model with three stages: (1) frame-level alignment using learned frame descriptors and soft matching to select reference candidates without trajectory synchronization, (2) patch-level correspondence with local correlation volumes and differentiable warping for geometric compensation without pose supervision, and (3) confidence-weighted fusion combining frame-level and patch-level confidence to suppress ambiguous matches before high-resolution decoding.

VSCDNet achieves 36.6 F1 on synthetic and 25.4 F1 on real-world data, outperforming all SCD and AOD baselines across all evaluation slices. The method is validated on a Stretch 3 mobile robot for visual surveillance (62.1% hit rate for abnormal events) and object incremental learning.

VSCD fundamentally redefines the input modality for scene change detection from image pairs to unconstrained video pairs, addressing a gap that no prior method in this wiki covers. GeSCF and C-3PO — evaluated as baselines — achieve 29.5% and 24.1% F1 respectively, showing that image-pair methods cannot handle the VSCD setting. The video-pair formulation is complementary to 3D-based approaches (VGGT-CD, O-SCD, SceneDiff) which require reconstruction; VSCD operates purely in 2D feature space without pose estimation or 3D geometry. The confidence-weighted multi-reference fusion shares conceptual DNA with O-SCD's self-supervised fusion but operates at the frame/patch level rather than pixel/feature level in 3DGS space. The benchmark fills a critical evaluation gap — prior video-based datasets (VDAO) had only 1 location; VSCD covers 218+ environments.
