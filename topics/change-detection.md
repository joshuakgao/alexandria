## Overview

# Change Detection — Overview

*Last updated: 2026-06-18 | Sources: 16 papers*

## Current thesis

Change detection is advancing along five complementary paths. **Path 1: Foundation model adaptation** (GeSCF, DynamicEarth) leverages off-the-shelf models (SAM, CLIP) for zero-shot generalization via task decomposition (segmentation → comparison → semantic grounding), optimized for satellite/natural imagery. **Path 2: Generative pre-training** (Changen2) synthesizes large-scale change data via stochastic modeling for remote sensing foundation models with inherent zero-shot capability. **Path 3: 3D geometric alignment** (SceneDiff) aligns multiview sequences in 3D space to eliminate viewpoint confounds, enabling instance-level change detection and tracking; training-free pipeline benefits naturally from model improvements. **Path 4: Online 3DGS-based detection** (O-SCD) uses a 3DGS reference representation with self-supervised fusion of pixel- and feature-level cues for real-time, multi-view consistent change detection — the first online method to surpass offline SOTA, closing a long-standing accuracy gap. **Path 5: Volumetric object-level tracking** (POCD) uses TSDF voxel differences with Bayesian object stationarity modeling for online map maintenance in semi-static environments — addressing the gap between static SLAM and dynamic SLAM for long-term robot deployment. **Path 6: LiDAR-based learned change detection** (Chamelion) uses a 4D sparse CNN with a dual-head architecture to jointly predict change class and cross-visibility confidence from LiDAR point clouds, enabling occlusion-aware online map maintenance in transient environments like construction sites. **Path 7: Hashing-based efficient detection** (HashSCD) encodes scenes as compact binary hash codes via patch-wise hashing with XOR aggregation, enabling 256x storage/compute reduction while maintaining competitive unsupervised detection accuracy — the efficiency-first approach for long-term, large-scale monitoring. **Path 8: Foundation-model joint inference for 3D registration** (VGGT-CD) exploits VGGT's joint-inference capability to resolve cross-temporal scale ambiguity without traditional SfM, producing high-purity 3D change maps from unposed multi-view images with coarse-to-fine registration and a monotonicity guarantee. **Path 9: Video-based change detection** (VSCD) redefines the input from image pairs to unconstrained video pairs, exploiting temporal continuity and multi-view geometry for sequence-level alignment without pose supervision — the first formulation targeting embodied agents that naturally capture environments via free-form video traversals. **Path 10: Text-conditioned change detection** (ViewDelta) resolves the fundamental relevance ambiguity by conditioning detection on natural language prompts, enabling a single model to be jointly trained across heterogeneous datasets and to serve different applications at inference time — the semantic control approach. These paths share the insight that labeled change data is scarce, but propose different solutions: adapt general models (Path 1), create domain-specific models (Path 2), exploit geometric alignment (Path 3), learn change representations online through self-supervised fusion (Path 4), learn from synthetically augmented single-session data (Path 6), compress representations to binary for scalability (Path 7), or leverage foundation model internals for implicit cross-temporal alignment (Path 8). The field is now branching along an efficiency axis (HashSCD's 256x compression) and an accuracy axis (VGGT-CD's 44-59% ATE improvement), with the fundamental question being whether these can converge.

## Key trends

**2022:**
- **Crowdsource-based change detection and map update.** CSCD (Wilf et al., Toga Networks + Huawei, 2022) presents a complete pipeline from VPS query to 3D map update — using regular localization queries as crowdsourced change evidence without dedicated mapping runs. The 5-DOF homography (preserving vertical lines for urban scenes) is more robust than standard 8-DOF. D2Net descriptor comparison within visual segments enables fine-grained change detection under viewpoint/illumination variation. Critically, the pipeline demonstrates that removing outdated features from the map improves both localization speed and accuracy — connecting change detection directly to downstream task performance. Anticipates RTMap's crowdsourced map improvement but at the point cloud / feature level.
- **Change detection meets visual question answering.** CDVQA (Yuan et al., TGRS 2022) introduces the first VQA formulation for multi-temporal remote sensing change detection. By allowing users to ask natural language questions about bitemporal images, CDVQA democratizes change information beyond expert users. The automatically generated dataset (122K+ QA pairs, 5 question types) built from semantic change maps establishes the task and a baseline framework with a change enhancing module. This interactive paradigm complements change captioning (which generates fixed descriptions) and later text-conditioned detection (which conditions pixel-level masks on text).
- **Probabilistic object-level change detection for map maintenance.** POCD (Qian et al.) addresses the semi-static gap — environments that change between robot traversals but not during continuous observation. The Bayesian object-level update rule (stationarity score × geometric TSDF change magnitude) enables online map maintenance without ghosting artifacts, foundational for long-term robot deployment in warehouses and industrial settings.

**2023:**
- **Remote sensing change captioning as a distinct task.** Chg2Cap (Chang & Ghamisi, TIP 2023) establishes that remote sensing change captioning (RSCC) requires domain-specific design: longer time intervals, seasonal/illumination distribution gaps, and compressed top-down views make natural image IDC methods insufficient. The hierarchical self-attention design (dual per-image self-attention → joint cross-image self-attention) with cosine-masked residual blocks achieves SOTA on Dubai-CC and LEVIR-CC, demonstrating that processing each time step individually before comparison yields better change features than direct feature subtraction or concatenation.

**2024:**
- **Two complementary paradigms for zero-shot change detection.** GeSCF achieves zero-shot via foundation model adaptation (SAM-based pseudo-masks and geometric matching); Changen2 achieves zero-shot via generative pre-training (synthetic change data generation). Both narrow the gap to supervised models (GeSCF 19.2% improvement, Changen2 3-10% gap), but through opposite mechanisms: adapting general models vs. creating change-specific models.
- **Synthetic data pre-training emerges as viable path for remote sensing.** Changen2 demonstrates that generative change models can create realistic, diverse multi-temporal datasets at scale (1.2M pairs); temporal diversity is the key to transferability. Challenges the assumption that supervised or contrastive pre-training is necessary; synthetic change data proves sufficient.
- **Self-supervised learning at scale.** Changen2's use of SAM-generated object contours enables training on massive unlabeled satellite imagery without manual annotation. Establishes that dense conditions (semantic masks) can be automatically extracted, removing annotation bottleneck for generative models.
- **Domain generalization and temporal consistency remain critical.** GeSCF establishes that in-domain vs. out-of-domain generalization has 90% error gap; Changen2's global dataset diversity (building change, LULC, disaster) ensures pre-trained models transfer across domain shifts. Both papers validate that change detection bottleneck is generalization, not supervised accuracy.
- **Benchmarking for generalization and transferability.** GeSCF introduces GeSCD benchmark with cross-domain metrics; Changen2 provides three large-scale synthetic datasets (15k, 27k, 1.2M pairs) and demonstrates pre-training transferability. Datasets become primary contribution alongside methods.

**2025:**
- **Open-vocabulary change detection unlocks new applications.** DynamicEarth combines SAM, SAM2, CLIP, and Grounding DINO to answer "4W" queries (when, where, what object, what change) in earth observation. Enables users to describe changes in natural language rather than selecting predefined categories.
- **Foundation model composition for complex reasoning.** Demonstrates that orchestrating complementary foundation models (segmentation → change comparison → semantic grounding) outperforms single-model approaches. Foreshadows future where change detection (and anomaly detection broadly) is assembly of foundation models rather than end-to-end training.
- **Domain diversity.** DynamicEarth focuses on satellite/earth observation imagery; GeSCF on natural scenes. Together, they span the application space from remote sensing to everyday photography, establishing change detection as a cross-domain problem requiring general solutions.
- **Text-conditioned change detection resolves relevance ambiguity.** ViewDelta (Varghese, Gao, Hoskere) introduces text prompts to specify what changes matter, enabling joint training across datasets with contradictory labeling conventions. A single model achieves competitive performance across street-view (PSCD), satellite (SYSU-CD), and urban (VL-CMU-CD) domains. CSeg dataset (500K+ pairs, 300K+ unique prompts) is the first large-scale text-conditioned SCD benchmark. Dramatically outperforms prior methods on unaligned image pairs (F1 63.6 vs. 28.4 on PSCD Diff-1) by removing spatial alignment assumptions from the segmentation head.
- **Frozen foundation model features as standard building blocks.** ViewDelta uses frozen DINOv2 (images) and SigLIP (text) embeddings, achieving 2x performance over trainable patch embeddings. Combined with GeSCF's use of SAM and DynamicEarth's CLIP, a clear pattern emerges: frozen pretrained features from diverse foundation models are becoming the standard substrate for change detection.
- **Image difference captioning as generalist VLM task.** OneDiff (Hu et al.) reframes IDC as a VLM task, using a siamese CLIP encoder with a Visual Delta Module (learnable delta tokens interacting with multi-level features via cross-attention) and a Coupled Sample Training strategy to bootstrap dual-image understanding from single-image pretraining data. DiffCap dataset (363K entries) combines retrieved real pairs, synthetic pairs, and existing benchmarks. Achieves up to 211% CIDEr improvement over prior specialist SOTA, demonstrating that generalist VLMs can dramatically outperform task-specific architectures.
- **3DGS-based object-level change detection.** 3DGS-CD (Lu et al.) introduces the first 3DGS-based method for detecting physical object rearrangements (moved/removed/inserted). Unlike scene-level methods, it produces object 3D segmentation masks and 6D pose changes — outputs directly useful for robotic manipulation. Works with as few as one post-change image; three orders of magnitude faster than NeRF-based methods. Bridges change detection and robotic manipulation by enabling workspace reset and 3DGS model update.

**2026:**
- **3D geometry as primitive for multiview change detection.** SceneDiff introduces 3D scene alignment as a core mechanism: static scene geometry from before/after sequences enables co-registration into shared 3D space, eliminating viewpoint confounds. Contrasts with pixel/semantic-only approaches; shifts the field toward considering geometric consistency as equal to appearance consistency.
- **Instance-level change tracking in real-world scenes.** SceneDiff focuses on concrete robotics/monitoring applications (room tidying, construction, disaster assessment) vs. earth observation focus of prior work (2024-2025). Introduces first instance-level multiview change detection benchmark with 350 video pairs and dense annotations; establishes new evaluation protocol for per-view and per-scene change assessment.
- **Training-free pipelines scale with model improvements.** SceneDiff demonstrates benefit of non-trainable composition: combines π³ (3D), SAM (segmentation), DINO (semantics) without any fine-tuning. Naturally improves as underlying pretrained models improve; removes need for task-specific training data or retraining. Validates hypothesis that foundation models enable training-free systems for structured geometric tasks.
- **Benchmarking shifts from satellite to general scenes.** GeSCF/Changen2 focused on remote sensing; SceneDiff covers 20 diverse indoor/outdoor categories. Indicates field broadening beyond satellite domain toward general vision; suggests that change detection principles may generalize across modalities (video, RGBD, etc.).
- **Online SCD closes the accuracy gap to offline methods.** O-SCD (Galappaththige et al.) is the first online method to surpass all prior offline approaches in F1/mIoU while maintaining >10 FPS. Demonstrates that the online/offline accuracy gap was a methodology problem (hard thresholding, heuristic fusion), not a fundamental limitation of incremental processing. Self-supervised fusion loss replaces MV3DCD's intersection heuristic with learned, multi-view consistent change inference.
- **Self-supervised fusion as alternative to foundation model composition.** While SceneDiff composes pre-trained models (π³, SAM, DINO), O-SCD uses a lightweight self-supervised loss to fuse pixel- and feature-level cues (SAM2-Tiny features + photometric error) into a persistent 3D change representation. Both are label-free but through different mechanisms: model composition vs. online self-supervised learning.
- **Efficient scene representation maintenance emerges.** O-SCD's change-guided selective reconstruction updates only changed regions, enabling 3DGS updates in seconds rather than minutes. This is critical for repeated inspections in robotics/monitoring; contrasts with continual learning approaches (CL-Splats, GaussianUpdate) that require longer optimization.
- **LiDAR change detection with learned occlusion handling.** Chamelion (Jang et al.) demonstrates that explicitly separating change classification from cross-visibility confidence estimation via a dual-head architecture dramatically improves robustness in occluded environments. Classical geometric methods (visibility, occupancy) suffer from high false positive rates in cluttered/occluded scenes; learning-based approaches (MapMOS, SPS) achieve good scan-wise detection but fail at map-level maintenance because they only classify where scan points exist. Chamelion's confidence-conditioned Bayesian update addresses both.
- **Single-session data augmentation for change detection training.** Chamelion's composition-based augmentation — extracting static objects via tracking, pasting into new locations to create pseudo changes — enables training without multi-session data collection. Parallels Changen2's philosophy of synthetic data generation but in 3D point clouds rather than 2D satellite imagery. Enables generalization across environments without per-dataset collection.
- **Cross-domain generalization in 3D.** Chamelion trained on one custom environment generalizes to different construction sites and the LiSTA indoor office dataset, demonstrating that the 4D sparse CNN + composition augmentation approach transfers across semantically diverse environments — a 3D analog of the cross-domain generalization challenge in 2D (GeSCF, Changen2).
- **Hashing-based representations for scalable SCD.** HashSCD (Duong et al.) demonstrates that binary hash codes can achieve competitive unsupervised change detection while reducing storage 256x and compute proportionally. Patch-wise hashing with XOR aggregation preserves spatial localization capability that global hashing lacks. The contrastive training at both patch and global levels enables unsupervised learning without change masks. Particularly relevant for long-term monitoring where storage and re-inference costs dominate.
- **Foundation model joint inference resolves cross-temporal scale ambiguity.** VGGT-CD (Zhang et al.) shows that feeding keyframes from both epochs into a single VGGT forward pass implicitly establishes a unified metric coordinate system, bypassing the registration-change paradox that defeats post-hoc alignment methods. The coarse-to-fine design with decoupled translation refinement and residual self-check provides a monotonicity guarantee — the fine stage provably never degrades the coarse result. Achieves 44% outdoor and 59% indoor ATE reduction on the WAT dataset while being 6-255x faster than registration baselines.
- **Efficiency vs. accuracy spectrum crystallizes.** HashSCD (2D, binary, 256x compression) and VGGT-CD (3D, continuous, highest accuracy) represent opposite ends of the efficiency-accuracy trade-off. O-SCD sits in the middle (real-time 3DGS-based). The field now offers practitioners clear choices along this spectrum depending on deployment constraints.
- **Video-based change detection emerges as a distinct paradigm.** VSCD (Yoon & Kim, ICML 2026) redefines the input modality from image pairs to unconstrained video pairs, formalizing how embodied agents naturally observe environments. The 1.1M-frame benchmark across 218 environments is an order of magnitude larger than prior video-based datasets. VSCDNet's three-stage architecture (frame-level alignment → patch-level correspondence → confidence-weighted fusion) achieves 36.6 F1 on synthetic data, outperforming all image-pair SCD baselines by 7+ points. The approach operates purely in 2D feature space without 3D reconstruction, complementing geometry-based methods (VGGT-CD, O-SCD, SceneDiff).
- **Sim-to-real gap persists as primary bottleneck.** VSCD's real-world F1 (25.4%) drops 11 points from synthetic (36.6%), despite training on diverse synthetic environments. This mirrors the domain generalization challenges seen across change detection (GeSCF, Changen2) and suggests that video-based methods face the same transfer bottleneck as image-based ones.
- **Confidence-weighted fusion as unifying principle.** Both VSCD (frame + patch confidence) and O-SCD (self-supervised fusion) independently converge on confidence-weighted fusion of multi-source change evidence — suggesting this is a fundamental design principle for robust change detection under alignment uncertainty.

## Open problems

- **Performance gap to fine-tuned models**: zero-shot GeSCF still lags fine-tuned methods by 2–3% on some benchmarks; trade-off between generality and task-specific performance unresolved
- **Foundation model selection**: DynamicEarth uses multiple models in parallel; unclear which are necessary, how to select optimal subset, or whether other foundation models could replace them
- **Semantic change understanding**: current methods focus on structural/geometric changes visible in pixels; semantic changes (e.g., object category shift without visual difference) not addressed
- **Multi-temporal reasoning**: most methods address pairs or single revisits; long-term multi-revisit monitoring with accumulated representation updates is nascent
- **Computational efficiency**: composing multiple foundation models is expensive; O-SCD achieves >10 FPS but requires pre-built 3DGS reference; HashSCD offers 256x compression but at some accuracy cost; VGGT-CD is fast for registration but requires VGGT inference per epoch
- **Binary vs. continuous representations**: HashSCD shows hashing is viable for SCD but the accuracy gap to continuous methods on noisy datasets (VL-CMU-CD) raises the question of whether hash codes can ever match continuous features for fine-grained change detection
- **3D registration without ground-truth evaluation**: VGGT-CD evaluates via trajectory accuracy (ATE/RTE) as a proxy for change detection quality; direct 3D change mask evaluation protocols are lacking
- **Evaluation protocols**: open-vocabulary change detection lacks standard benchmarks; online SCD evaluated only on PASLCD (room-scale); building/city-scale evaluation missing
- **Real-world applicability**: methods span satellite (DynamicEarth), natural scenes (GeSCF), and room-scale 3D (SceneDiff, O-SCD); applicability to other modalities (hyperspectral, SAR, thermal) unclear
- **Dynamic scenes during observation**: O-SCD assumes static scene within each revisit; handling moving objects during observation remains open
- **Continuous representation maintenance**: O-SCD updates representations post-hoc; truly continuous, online representation updates during observation remain unsolved
- **Video-based sim-to-real transfer**: VSCD trained on synthetic data drops 11 F1 points on real-world evaluation; domain adaptation, few-shot real-world fine-tuning, or self-supervised pretraining on real video could close this gap
- **Unified 2D video + 3D geometry**: VSCD operates in 2D feature space; VGGT-CD/O-SCD operate in 3D; whether combining temporal video matching with 3D reconstruction yields better results than either alone is unexplored
- **Text conditioning + real-world evaluation**: ViewDelta's CSeg is synthetic; no real-world dataset with diverse text prompts and varying views exists. Real-world text-conditioned SCD evaluation remains an open challenge
- **Discovery vs. specification**: ViewDelta requires users to specify what to look for; cannot discover unexpected changes unless "all changes" prompt is used. Whether text conditioning can be combined with anomaly-detection-style discovery is unexplored
- **Caption-level vs. mask-level change description**: OneDiff generates natural language descriptions of differences between image pairs but does not spatially localize changes; integration of captioning with pixel-level detection remains open

## Contradictions and debates

- **Zero-shot vs. fine-tuned trade-off**: GeSCF's generality comes at cost of absolute performance; whether users prefer robust-but-slightly-worse zero-shot or brittle-but-accurate fine-tuned models depends on deployment constraints (which are often unstated)
- **Temporal consistency enforcements**: GeSCF uses geometric matching; whether alternative approaches (learned consistency, contrastive objectives) could achieve better results is unexplored
- **Foundation model composition strategy**: DynamicEarth's M-C-I framework is intuitive but not the only possible decomposition; whether alternative pipelines (end-to-end with foundation model backbones, hierarchical composition) might be better is unclear
- **Online vs. offline value proposition**: O-SCD shows online can match offline accuracy; whether offline methods still have advantages in specific settings (very large scenes, extreme viewpoint changes) remains to be demonstrated
- **Self-supervised vs. model composition**: O-SCD's learned fusion vs. SceneDiff's pre-trained model composition represent fundamentally different philosophies for label-free SCD; unclear which scales better to new domains
- **RGB vs. LiDAR change detection**: Chamelion, O-SCD, and SceneDiff address the same fundamental problem (what changed between observations?) but in different modalities. LiDAR provides direct 3D geometry but no appearance; RGB provides rich appearance but requires 3D reconstruction. Whether unified multi-modal change detection (RGB + LiDAR) would outperform modality-specific approaches is unexplored
- **Point-level vs. object-level change tracking**: Chamelion operates at the voxel level while POCD and SceneDiff track at the object level. Object-level approaches enable richer downstream reasoning (what moved where) but require reliable object segmentation; point-level approaches are more general but lose semantic structure
- **Joint inference vs. post-hoc registration**: VGGT-CD's joint-inference approach fundamentally changes the registration paradigm by establishing scale priors before dense reconstruction, while traditional methods attempt to recover alignment post-hoc. Whether joint inference scales to many-epoch temporal sequences or large-scale scenes is unclear
- **Compact binary vs. rich continuous for long-term monitoring**: HashSCD argues for extreme compression (8 KB/image); VGGT-CD and O-SCD argue for rich 3D representations. The optimal point depends on deployment constraints (edge devices vs. cloud, revisit frequency, required change granularity)
- **Image pairs vs. video pairs as input**: VSCD demonstrates that video pairs substantially outperform image-pair methods (36.6 vs. 29.5 F1) under realistic conditions; but video requires more storage, compute, and temporal reasoning. Whether the extra information in video is always worth the cost, or whether sparse multi-view sampling (as in VGGT-CD) suffices, is an open question
- **Text-conditioned vs. detect-then-describe**: ViewDelta conditions detection on text (specify first, detect second); DynamicEarth detects all changes then describes them (detect first, describe second). These represent opposite information flow directions; unclear which is more useful in practice, and whether they can be unified
- **Specialist vs. generalist IDC**: OneDiff demonstrates that a single VLM-based model can dramatically outperform specialist IDC models across diverse benchmarks (surveillance, editing, fine-grained biology). Whether this generalist advantage holds for domain-specific deployment (e.g., medical image comparison) or whether specialist architectures still win in narrow domains is untested
- **Captioning vs. VQA for change understanding**: Chg2Cap and OneDiff generate complete change descriptions; CDVQA answers specific questions interactively. Both have merits — captioning provides comprehensive summaries while VQA enables targeted queries. Whether a unified model can do both well (generate full descriptions and answer specific questions) is unexplored
